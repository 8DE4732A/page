Mozilla Firefox 112发布 减少英特尔CPU上的GPU使用、改进密码字段等

Mozilla Firefox 112现在已经在正式版频道中发布。Mozilla在这一版本中提供了浏览器的最新功能更新，还减少了在装备英特尔处理器的PC上的GPU的使用，略微改进的密码字段，增强用Ctrl + Shift + T快捷键恢复已关闭会话的能力，以及更多。

以下是Mozilla的官方更新日志：

右键单击密码字段现在可以显示一个选项来显示密码。

Ubuntu Linux用户现在可以从Chromium Snap软件包中导入他们的浏览器数据。目前，这只有在Firefox没有作为Snap包安装的情况下才会起作用，但正在努力解决这个问题。

你是否使用标签栏中的标签列表面板？如果是的话，你现在可以通过中键单击该列表中的项目来关闭标签。

现在可以通过使用（Cmd/Ctrl）-Shift-T来取消关闭一个标签，如果在同一会话中没有更多关闭的标签需要重新打开，同样的快捷键将恢复前一个会话。

对于所有设定ETP严格级别的用户，新版本扩展了从URL中删除的已知跟踪参数列表，以进一步保护我们的用户免受跨网站跟踪。

在Windows的英特尔GPU上实现了软件解码视频的叠加。提高了视频缩小的质量，减少了GPU的使用。

各种安全问题的修复。

被废弃的U2F Javascript API现在被默认禁用。U2F协议仍然可以通过WebAuthn API使用。U2F API可以使用security.webauth.u2f首选项重新启用。

清理按钮被添加到日期选择器面板，允许用户快速清除日期或日期-时间-本地类型的输入，并提供一个跨浏览器的熟悉体验。
