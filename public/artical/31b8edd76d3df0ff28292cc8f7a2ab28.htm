PC Game Pass又在全球40个国家推出

近日微软又在全球40个国家新推出了PC Game Pass。今年2月，微软曾在这些国家推出了PC Game Pass预览项目，允许注册了Xbox Insider项目的玩家以首月“特别测试价格”试订阅PC Game Pass。

这样做的玩家将免费获得两个月的PC Game Pass会员资格，无需进一步操作。这将在未来几天在会员帐户中提供。

微软表示，新会员还将在前三个月获得“特别欢迎优惠”，期限有限。
