三款索尼新品耳机发布：仅重33g 轻若无物

2月28日，索尼发布了三款耳机新品：一款为非入耳开放式运动耳机Float Run，以及两款高性价比头戴式耳机——索尼史上最轻头戴式无线降噪耳机WH-CH720N，以及主打多彩莫兰迪配色、兼具时尚和品质之感的WH-CH520。

Float Run

索尼打造的运动耳机Float Run，重量仅约33克，让用户的佩戴体感轻若无物。音质表现上，新品采用了16毫米驱动单元，得益于索尼深厚的声学积累与专业调音技术，营造出宽广的立体声场、三频清晰、低音澎湃的聆听体验。

Float Run的发声单元悬于耳道旁，这种开放式设计，使用户的音乐体验更加自在通透。IPX4防水等级、约10小时续航与快充等功能，让用户享受轻松便捷、安心放心的音乐体验。

相比于入耳式与头戴式耳机，这款非入耳开放式耳机Float Run整体轻量化及耳挂后挂的精巧设计，使佩戴体验轻松便捷、轻盈舒适。

发声单元悬浮在耳道旁，不会对耳道造成挤压，避免对耳道的损伤；天然的散热属性也让Float Run保持干爽透气。

值得注意的是，Float Run不会产生骨传导耳机常有的头骨震动感，在佩戴感受上更加贴合多元运动场景。

Float Run采用了长度合理且柔韧的后挂，以及拥有宽接触面的耳挂。在二者的协同作用下，分散了耳部压力，同时也带来合适的夹持力，使用户的佩戴体感既舒适又稳固。

这款耳机能够轻松适配多种头型、耳形，用户还可以根据自己的喜好和需要，调整耳挂的倾斜角度，确保穿戴舒服，并且在跳跃、奔跑、伸展等伴随剧烈摇摆的运动场景中，依旧能够保持耳机佩戴稳固。

质感颇高的耳挂与后挂设计，也增添了Float Run的可穿戴性，不同发型的用户可以安心佩戴，爱戴帽子与发带的运动达人们也可放心入手，穿戴无负担，运动更自由。

Float Run传承索尼深厚的音频技术和产业优势积累，兼顾开放式与好声音，让用户在运动时也充分享受好音质的聆听体验。

新品搭载16mm驱动单元，秉承索尼经典的调音特色，呈现出丰富的声音细节、层次清晰的三频和澎湃的低音。

与骨传导的发声原理不同，非入耳开放式设计能温和自然地糅合左右声道，打造更为宽广的立体声场，突显自然开阔的空间感。

Float Run支持IPX4防水，用户不用担心汗水或雨水侵袭；拥有约10小时的续航、约200小时的待机时长，支持快充，充电10分钟可使用约1小时，续航无焦虑，运动更有力。

Float Run也具备高清语音通话的功能，用户通过耳机上的按键即可实现快捷操作。拥有多项功能的Float Run也非常适合人们在工作等日常场景下使用。

此外，索尼继续践行科技环保，耳机的外包装盒采用了不含塑料的环保材质，让运动与音乐体验更加环保可持续。

另外，此次一同发布的还有两款头戴式耳机——WH-CH720N与WH-CH520，其中WH-CH720N集智能降噪、高品音质及长效续航等功能于一身。

WH-CH520拥有多彩莫兰迪配色、具备好音质与高清通话等多元功能，这两款耳机凭借高性价比让入门级用户也可以感受到由索尼打磨的品质体验。

售价方面，非入耳开放式运动耳机Float Run售价999元；头戴式降噪耳机WH-CH720N售价799元；头戴式耳机WH-CH520售价499元，从今日起逐步开启预售。
