QQ Linux 3.1.1版发布：底层重写、支持国产自主龙芯CPU

2月24日，QQ Linux 3.1.0版本发布，采用全新架构，基于QQ NT技术架构驱动，最重要的是正式支持龙芯中科自主指令集的LoongArch龙架构平台。新版可在基于龙芯3A5000处理器的UOS、麒麟、Loongnix等系统上原生运行，与x86、Arm并列曾为QQ官方支持的三大CPU架构。

3月29日，QQ Linux升级为最新的3.1.1版本，变化还不小。

新版新增了四大功能，包括可跟随系统的夜间模式、QQ截图、群应用、全局搜索，可用性体验进一步提升，和Windows下也越来越接近。

此外，新版还修复了部分已知Bug，但未详细披露。

QQ Linux 3.1.12023.03.28更新日志：

1、新增夜间模式并支持跟随系统，享受更沉浸的聊天体验；

2、新增QQ截图能力，快捷方便，提升办公效率；

3、新增群应用功能，图片、文件轻松传送，群聊更高效；

4、新增全局搜索能力，进一步提升内容搜索效率。

5、修复若干已知Bug。

QQ Linux版目前支持x64（x86_64/AMD64）、arm64（aarch64）、LoongArch三种架构，每种架构又可支持Debian系、Red Hat系、其它发行版中的一种或几种（未来可能继续扩充），每一次发布均会提供架构和发行版的若干种组合支持的安装包，可按下面所述的规则进行选择。

每一个安装包会按照形如如下的格式命名： 

linuxqq_3.0.0-100_x86_64.rpm 

[产品名]_[版本号]_[架构].[格式]

安装命令示意：

RPM

sudo rpm -ivh linuxqq_3.0.0-100_aarch64.rpm 

DEB

sudo dpkg -i linuxqq_3.0.0-100_amd64.deb 

或

sudo apt install -y linuxqq_3.0.0-100_amd64.deb 

AppImage

chmod +x linuxqq_3.0.0-100_x64.AppImage 

./linuxqq_3.0.0-100_x64.AppImage

卸载命令示意：

RPM

sudo rpm -e linuxqq 

DEB

sudo dpkg -r linuxqq
