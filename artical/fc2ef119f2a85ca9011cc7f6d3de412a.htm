三星Galaxy Tab S9 Ultra细节泄露 尺寸、重量、配置曝光

据传，三星将在Galaxy Tab S9 Ultra上"全力以赴"，而一位消息人士提供的最新规格可能证明，该公司正在准备一个iPad Pro级别的竞争对手，二其硬件与笔记本电脑相当。以下是你需要知道的关于即将推出的旗舰平板电脑的一切。

泄漏的规格信息显示，Galaxy Tab S9 Ultra可以一定程度上取代笔记本电脑，从尺寸开始，Ice Universe发布的推文称Galaxy Tab S9 Ultra的尺寸为326.4 x 208.6 x 5.5毫米，与Galaxy Tab S8 Ultra相比不会有变化，显示屏的对角线尺寸为14.6英寸，分辨率为2960 x 1848，重量737克，支持IP68防尘防水。

与Galaxy Tab S8 Ultra相比，屏幕面板本身似乎也没有什么不同，这表明除了可能调整颜色、亮度、电源效率和其他属性外，三星可能在Galaxy Tab S9 Ultra上使用相同的面板。如果是真的，它可以帮助三星避免了生产成本增加。另外，Galaxy Tab S8 Ultra一开始就已经足够大，所以没有必要再扩大这个显示屏尺寸。

据称，两款平板电脑的11200mAh电池也是一样的，尽管之前有传言提到Galaxy Tab S9 Ultra将采用更小的电池，由于据传这款巨大的平板电脑将配备高效的骁龙8 Gen 2，也就是Galaxy S23系列使用的那一款，CPU和GPU的时钟速度略高。鉴于Galaxy Tab S9 Ultra的尺寸，给SoC降温毫不费力，并且相比上一代电池续航大概率会有所提升。

最后，Galaxy Tab S9 Ultra支持16GB的LPDDR5X内存，当前一代Galaxy Tab S8 Ultra也提供了16GB内存，但它相比新一代内存的速度较慢，耗电量也较大。然而，三星不太可能为Galaxy Tab S9 Ultra提供16GB内存选项作为标准配置，客户将需要多支付费用以获得16GB内存。

假设这些泄露的规格是真实的，那么三星Galaxy Tab S9 Ultra实际上是在上一代机型基础上的小幅升级，而选择它的消费者主要会被芯片和内存的升级所吸引。
