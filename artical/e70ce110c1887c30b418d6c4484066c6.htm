媒体创建工具的替代方案使下载Windows 10和11的ISO变得更加容易

获取用于全新安装或实验的官方版 Windows ISO 很容易：只需从各自的网站下载适用于 Windows 10 或 11 的媒体创建工具应用程序。 然而，官方程序有一些明显的缺点，例如有限的版本可用性或架构依赖性。不过现在有了解决方案，您可以使用 Michael Niehaus 提供的新 MCT 替代方案绕过这些限制。

Oofhours Media Tool使用的是来自微软的官方清单，这些清单指引了镜像存储在微软公司的服务器上。 该应用程序下载的是 ESD 文件，然后将它们转换为随时可用的 ISO。

OMT 允许选择 Windows 10 或 11、体系结构（x64、x86 或 ARM64）、语言和版本。 除了标准的 Home 和 Pro 之外，该应用程序还可以轻松获得非主流 SKU，例如 Enterprise、Education 或 Professional for Workstations。 请注意，虽然该应用程序按照清单列出了一些不存在的版本（ Windows 10 Cloud 之类），因此在下载特定版本时会遇到错误。

与官方媒体创建工具应用程序不同，OMT 无法将下载的图像直接制作成 USB 启动盘，但可以通过打开 ISO 并将其内容复制到 FAT32 格式的 USB 来解决此问题。 或者，使用其他实用程序，例如最近更新的 Rufus是一个更加强大的工具。
