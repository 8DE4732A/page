StatCounter：Windows 10份额跌破70% Windows 11继续缓慢攀升

StatCounter发布了2022年11月的报告展示了关于操作系统和浏览器的最新数据。根据调查结果，在2022年11月，Windows 10市占率首次跌破70%。虽然微软的这一版操作系统仍然是王者，但随着后继者变得更受欢迎，其市场份额正在慢慢减少。

StatCounter称，Windows 10现在位于69.77%的位置，与前一个月相比下降了-1.49%。

Windows 11没有打破任何升级速度纪录，但它仍然在保持相对缓慢但稳定的步伐以尝试提升自己的地位。2022年11月的报告显示，Windows 11目前的市占率为16.12%，与2022年10月相比，变化为+0.67%。

Windows 7仍然是第三大最受欢迎的操作系统，市场份额与Windows 11相对接近--约为10.24%（+0.62）。微软计划在明年年初结束Windows 7的扩展安全更新计划，因此预计该操作系统的市场份额会下降得更快一些。然而，许多客户特别是企业客户更愿意继续使用Windows 7，这就是为什么一些开发商正在考虑延长他们的软件支持。

至于Windows 8.1和8，这两个操作系统分别有2.54%和0.79%（+0.09和+0.1），Windows XP，这个本世纪初就诞生的操作系统仍以0.4%的市场份额保持存在。

尽管Windows拥有超过15亿台活跃设备，但它并不是最受欢迎的移动操作系统。这一殊荣由Android系统保持，其市场份额为43.37%（-1.13%），尽管Windows以29.24%（-0.93%）排名第二，而iOS以17.25%（-0.32%）排名第三。
