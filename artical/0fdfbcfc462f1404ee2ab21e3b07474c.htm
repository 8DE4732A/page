Windows 11今年更新内容泄漏：现代音量混合器、实验工具和新的文件管理器

Windows 11将在2023年晚些时候进行另一次大更新。虽然我们已经对操作系统的下一步有了很好的了解，但预览版的发布说明让我们知道看微软正在研究最近在测试版本中发现的一些新的令人兴奋的功能。

一个新的音量混合器

改变单个应用程序和浏览器音量的最快捷方式是点击音量图标，打开音量混合器。Windows 11的音量混合器只能通过设置应用程序访问，它可以让你改变任何打开的浏览器或应用程序的音量，使特定的应用程序更响亮。

值得庆幸的是，微软终于在测试一个新的现代音量混合器，可以直接通过任务栏访问，类似于Windows 10的音量混合器。

微软还没有公开谈论这个新的音量混合器，它被隐藏在代码中。正如你在上面的截图中所看到的，你可以快速调低或增加已经打开的应用程序的音量，同时使Spotify等音乐应用程序的音量更大。当然，也可以将任何应用程序静音，使另一个应用程序或系统声音更大，这个想法是为了让Windows 10内的声音管理更容易。

Windows 11的新音量混合器是隐藏的，公众无法测试它。这是因为它仍在开罚中，微软还不准备谈论它。

实验性工具：一种解锁功能的新方法？

微软正在测试Windows Update -> Windows Insider Program页面内的一个新选项，名为"AllowExperimentalFeatures"。

顾名思义，看起来AllowExperimentalFeatures最终可以允许Windows Insider Program中的测试者打开被公司禁用的功能。

Windows Insider Program 页面正在获得一个"实验性功能"列表

实施工作尚未完成，但这有可能成为一种官方方式，在A/B测试不够的情况下切换隐藏在操作系统中的功能。

新的文件资源管理器与微软365有更深的联系

Windows 11也将获得一个新的文件资源管理器，它有一个基于XAML的主页和Microsoft 365的集成。

新的主页紧跟Windows 11的风格，地址栏和搜索栏的边角更加圆滑，重新设计的文件资源管理器更像微软365仪表板或网络浏览器，而不是传统的文件管理器。

文件资源管理器中的新侧窗格

一个新的"推荐文件"选项卡将被加入，当选择文件时，它使用Microsoft 365来推荐类似的文件。文件资源管理器还可以显示电子邮件会话、评论、伙伴之间的对话以及文件何时被同事共享。

据消息人士称，微软正在考虑将微软365的功能引入文件浏览器。使用现有的外壳是不可能做到的，这就是为什么新的文件资源管理器使用基于网络的XAML设计，允许用户在文件管理界面中直接访问微软365的内容，并通过一个名为"标签"的新功能来组织所有内容。

微软仍在开发新的文件资源管理器，计划可能会有变化，但微软365更深入地整合到Windows 11中似乎是该公司的长期计划。
