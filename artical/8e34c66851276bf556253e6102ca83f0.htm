特斯拉在美德降价 国外网友用孙子兵法“批评”马斯克

北京时间1月13日消息，距特斯拉在中国降价一周后，特斯拉又于今日宣布在美国对全系车型进行全面大规模降价，并在德国调降MODEL 3价格。对于特斯拉此次降价，也引发了不少外国网友的热议，有网友联想到了中国的《孙子兵法》，他认为马斯克此举是为了“消灭”传统汽车行业。


在美国，其中降幅最大的是基础版的Model Y，降幅为13000美元(约合人民币8.7万元），其他Model系列也均有不同幅度的降价。在德国，Model 3后轮驱动版售价降至43990欧元(约合人民币32万元），Model 3长续航版售价元降至53990欧元(约合人民币39万元），Model 3高性能版售价降至60990欧元(约合人民币44万元）。

在国外社交媒体平台Twitter、油管上，很多网友对特斯拉此次降价进行讨论。有网友联想到了中国的《孙子兵法》，他认为马斯克此举是为了“消灭”传统汽车行业。


部分网友对特斯拉此次降价持反对意见，他们发文和评论称，作为特斯拉车主，此次降价让他们“损失惨重”，他们感到非常愤怒、不满与失望。





也有部分国外网友对特斯拉此次降价表示认可，他们认为这将有利于缓解电动汽车市场的经济低迷，并且自己也有意向购买降价后的特斯拉汽车。

1月6日，在中国，据特斯拉官博，特斯拉国产车型大幅降价，Model 3起售价22.99万元，Model Y起售价25.99万元，交付周期均为1-4周。

具体降价幅度：

Model 3降价3.6万，售价22.99万；

Model 3 高性能降价2万，售价32.99万；

Model Y降价2.9万，售价 25.99万；

Model Y长续航降价4.8万，售价30.99万；

Model Y高性能降价3.8万，售价35.99万。
