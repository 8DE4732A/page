这款来自小厂的迷你主机 把小米新品按在地上摩擦

前阵子小米不是发布了一台迷你主机么？它搭载英特尔酷睿 i5-1240P 处理器，16GB DDR4 内存以及 512GB PCIe 4.0 SSD，首发价 3699 元：同时，小米还推出了这款机型的准系统版本，也就是在前面这配置的基础上，去掉了内存、固态以及 Windows 11 系统，价格便宜 900，售价 2799。



不夸张的说，第一眼看到这款产品，当时我就种草了。

因为对于像我这种平时不怎么玩游戏，只拿电脑刷剧办公的人来说，这种形态的主机简直就像是为我量身定制，重量轻体积小一点都不占地。


然而等到我做功课的时候才发现，网友们对于这款产品的风评却不咋滴。

一方面是因为有人把这款主机拆了之后，发现它的内部构造几乎跟英特尔之前发布的一款 nuc 一模一样；另外 i5-1240P 的性能也差点意思。


俗话说的好没对比就没有伤害，在搜索同类产品之后，我承认之前确实是孤陋寡闻了，原来市面上做这类迷你主机的厂商还真不少……

而且你敢信 3699 这个价位，我们其实已经能买到搭载 AMD R7-6800H 处理器的迷你主机！


尽管 i5-1240P 的核心数更多（ 4 大核 + 8 小核 / 16线程），但是局限于频率和功耗，它的多核性能反而不如 R7-6800H（ 8 核 / 16 线程 ）。

再加上 6800H 搭载的 680M 集显，它的性能足以媲美 MX550 或 1050Ti 级别的独显，所以价格差不多的情况下这俩处理器是我肯定首选 AMD~

而在这些搭载 R7-6800H 处理器的迷你主机中，零刻 SER6 Pro 应该算是被网友们提及最多的机型之一。


3688 的价位，它除了搭载 R7-6800H，还给了我们 32GB DDR5 内存以及 500GB PCIe 4.0 SSD，并且也支持 WiFi 6，性价比可以说吊打小米。

当然了，要是你选择的是没有内存和硬盘的准系统，那么 2558 就能获得一台迷你主机，比标准版直接便宜一千，说实话这价格我心动了！


最终通过层层关系，托尼也是成功搞来一台零刻 SER6 Pro，接下来就带大家一起看看它使用起来到底咋样（ 全程无广，可放心食用 ）。


老样子咱们先从外观说起。

有一说一本来我对它的做工没抱多大期待，因为零刻算不上一线品牌，再加上它的价格摆在这里，还要啥自行车？没曾想它的做工竟然还可以。


主机外壳采用了全金属材质，顶部则是使用了一块透气织物装饰板，在它的右下角还印有零刻品牌的 logo 标识，同时四周还做了一圈亮面切割。


再加上它墨绿的配色和方正的造型，这样的颜值还蛮符合理工科的审美好吧，说好看可能差点意思，但是绝对不丑。

主机整机五百多克的重量相对来说也比较轻，但要是再加上这块板砖适配器，它的重量就来到了一千多克，如果壁挂还得考虑电源的摆放位置。


不过根据官方给出的说法，零刻已经联合航嘉制作了新的 120W 电源，体积相较之前缩小了 2/3，后续可以免费更换，这点绝对要给好评。


另外得益于标准版自带 Windows 11 专业版系统，所以这台主机开机就能用，你还别说，给它连上蓝牙键盘和鼠标，放在桌面上确实挺简洁：


要是不嫌麻烦，你甚至可以把它装包里每天带到公司，下班后再带回家。

这样无论生活还是工作，用的都是同一台电脑，数据都保存在同一台电脑里，顺便还能薅一波公司每个月几百的设备补贴，如果有的话~


接口方面。

在零刻 SER6 Pro 机身的背面设有 1 个 2.5G 有线网口、1 个 USB 3.0、1 个 USB 2.0 和 2 个 HDMI 2.0 接口，最左边的电源插孔仍然是 DC 口：


而在它的正面则设有 2 个 USB 3.0、1 个 USB 4.0 接口和 1 个 3.5mm 耳麦接口：


这样的接口配置怎么说呢，对我来说满足日常使用是够了，不过相比小米那款迷你主机，它少了一个 Type-C 接口，多少让人觉得有点膈应~


这里值得一提的是，零刻 SER6 Pro 的这个 USB 4.0 接口除了支持供电、数据传输（ 20Gbps ）和视频输出外，还支持外接显卡坞。

不过我寻思着我都买迷你主机了，再给它外接一个独显好像貌似也没那个必要，不然我为啥不直接整台主机呢？要知道显卡坞可一点都不便宜。


当然接口有这功能肯定比没有好。

怀着好奇的心理，托尼也是试着拆了下机，发现它内存条使用的是两条 16G 英睿达的 DDR5 内存，而硬盘则是金士顿的 PCIe 4.0 硬盘。

与此同时这台迷你主机还提供了一个额外的硬盘位用于扩展，只不过这个硬盘位只支持 2.5 英寸的硬盘。


考虑到这款迷你主机不支持内存超频，内存的上限就摆在那里，上 64G 内存又没那必要，所以留给我们自由发挥的大概也就只剩硬盘。

实测，标配的硬盘读写速度属实不咋滴，建议有条件可以直接上更好的 PCIe 4.0 固态，或者直接买准系统版，自己再另外配内存和硬盘：


至于差友们最关心的性能方面。

作为 CPU 厂里最会做显卡、显卡厂里最会做 CPU 的厂家，AMD 这次可谓完美结合了两者的优势，推出的 R7-6800H 绝对堪称一款神 U。

它不仅可以提供充足的处理器性能，更重要的是还有一颗规格高到离谱的核显 Radeon 680M，这颗核显理论跑分甚至比 GTX 1650 还高：


反馈到实际游戏体验上，在 1080P 分辨率下，这台迷你主机也不是不能玩 3A 游戏，当然这个时候画质和帧率我们就只能二选一：


有些差友可能还会关心它的散热和运行噪音。

就这么说吧，在 AIDA64 软件中，单烤 CPU，它的主频最高可以到 3.8GHz，最低 3.2GHz，CPU 温度维持在 88℃ 左右，表现只能说一般。


噪音方面，风扇全开情况下，这台主机的噪音表现对我来说完全能接受，就像旁边摆了台小风扇，如果是在办公司，基本可以忽略。

总的来说，这种形态的主机注定不是适合所有人。


迷你主机的使用场景比较固定，要便携可以买笔记本，要性能可以自己组台式机，它最吸引人的点，大概就在于紧凑的机身可以节省更多空间。

至于 “ 空间 ” 在大家心里值多少钱，就得因人而异了。

在我看来这些厂商都要感谢下 AMD，感谢 AMD，让迷你主机得以用上更强的核显，现在不少轻薄本和游戏掌机用的也是 R7-6800H。


还能说啥呢，AMD YES！
