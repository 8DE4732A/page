Windows 11 KB5022364 (22H2)预览版给操作界面带来了更多圆角

Windows 11 KB5022364（版本22H2）更新现在可供测试者在Beta频道使用，它带有几个明显的改进。例如，微软为"开始"菜单的搜索栏带来了更多的圆角，修复了几个错误等等。

对于那些在生产版22622（22H2）上的用户，他们将通过一个启用包获得Build 22623。按照微软的说法，Windows启用包只是将构建号增加了一位数字，由于隐藏的功能标志和注册表，功能会自动打开。

然而，微软官员指出，该方法被用于Beta通道，公司可能选择在启用包更新之外发布Build 22623的功能。

KB5022364并不是一个大版本，但它带有一些明显的调整。例如，"开始"面板内的搜索栏现在具有圆角。这一变化是为了反映任务栏上新搜索框的设计。对于那些不知道的人来说，微软最近开始推崇任务栏的新搜索设计。

KB5022364中改进和修复的内容

如上所述，在这个预览更新中没有太多明显的变化，但如果你使用快速助手，你会注意到它已经从Windows工具中移出，可以直接通过开始菜单的所有应用程序列表中自己的条目访问它。换句话说，微软正在使它更容易找到。

微软还修复了一个问题，即任务栏的上半部分可能被剪掉，"卸载应用程序对话框"可能无法在开始菜单中正确呈现。这个错误使按钮在某些情况下无法使用，但现在终于在最新的更新中得到了修复。

开发团队还重新设计了黑暗模式和搜索栏的任务管理器，正在得到几个错误的修复。例如，已经修复了一个发布者的名字在进程页面上没有正确匹配的bug，另一个bug会让一些服务不显示在服务页面，问题也已经解决。

Windows 11 KB5022364任务管理器

任务管理器修复了一个错误，即一旦应用了主题更改，进程页面的内容区域就会闪烁，当试图结束某些进程时，任务管理器还会崩溃。

最后，微软正在为预览版.NET框架更新引入新的体验。所有未来的预览版.NET框架更新现在可以通过设置>Windows更新>高级选项>可选的更新来管理。
