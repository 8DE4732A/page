英特尔Meteor Lake-S台式机CPU传闻被取消 LGA 1851可能支持三代酷睿

OneRaichu带来了关于英特尔酷睿桌面CPU家族的新传言，他说Meteor Lake可能不会出现在LGA 1851插座上。之前有传言说，英特尔可能会取消Meteor Lake CPU的桌面发布，现在看起来这些可能会成为现实。

根据OneRaichu的说法，他在过去的泄密事件中一直非常准确，泄密者称Meteor Lake-S桌面CPU可能不会在桌面平台上发布，但仍将走向移动领域。几个月前，我们报道了Meteor Lake-S的原始计划，其中包括桌面LGA 1851平台的各种SKU。

该SKU列表包括：

Meteor Lake-S 22 (6P + 16E) / 4 Xe Cores / 125W TDP

Meteor Lake-S 22 (6P + 16E) / 4个Xe核心 / 65W TDP

Meteor Lake-S 22 (6P + 16E) / 4个Xe核心 / 35W TDP

Meteor Lake-S 14 (6P + 8E) / 4个Xe核心 / 65WTDP

Meteor Lake-S 14 (6P + 8E) / 4个Xe核心/ 35WTDP

但随着英特尔Meteor Lake-S据称被取消，看起来我们不得不修改我们所得知的英特尔台式机核心阵容。因此，从2023年开始，英特尔计划推出Raptor Lake Refresh CPU系列，它将以更高的核心时钟和优化的工艺取代现有的芯片，以实现更好的电力输送。最新的路线图证实了这一点，它们将与所有现有的LGA 1700/1800插座主板兼容。这将意味着目前的插座将持续三代CPU家族，这在近年来都很罕见。

但是LGA 1700/1800不会是唯一能持续三代的插座。有传言称，下一代LGA 1851插座也可能支持至少三代CPU。我们可以期待在下一个插座上看到的三个系列将必须是Arrow Lake-S（ARL-S），可能存在的Arrow Lake Refresh和Panther Lake (PNL-S)。

英特尔第15代Arrow Lake-S桌面CPU将恢复我们今天在Raptor Lake CPU上得到的24个核心。Arrow Lake-S顶级芯片将提供多达24个核心，这将是8个性能核心和16个效率核心的组合。

根据以前的泄漏，该阵容将只有酷睿i7和酷睿i9两种选择。CPU将保留英特尔4（CPU）+台积电N3（GPU）的SKU节点布局。据传，20A节点不会出现在台式机阵容中。英特尔Arrow Lake-S桌面系列预计将有以下SKU。

Arrow Lake-S 24（8P+16E）/4个Xe核心/125W TDP

Arrow Lake-S 24 (8P + 16E) / 4个Xe核心/65W TDP

Arrow Lake-S 24 (8P + 16E) / 4个Xe核心/ 35W TDP

从每个插座支持的两个系列切换到三个系列，似乎是英特尔保持桌面战场竞争激烈的一个好举措，特别是与AMD的做法相比可以挽回一些老主板用户的心。另据报道，预计将带来大规模性能和IPC增长的Royal Cove核心架构将不会出现，直到Panther Lake之后的一代，预计是Nova Lake，那是2025年以上的产品。

我们从以前的报告中知道，英特尔正在准备一个被称为"V"的新插座，它将为至少两代桌面CPU、Arrow Lake-S及其更新和未来产品提供支持。这个LGA 1851插座的尺寸将与现有的LGA 1700/1800插座非常相似，但将提供更多的引脚，并增加对新的/增强的功能的支持。

在最近的投资者电话会议上，英特尔表示，他们在英特尔4和英特尔3产品上的进展非常好。第14代Meteor Lake的生产步骤计划在本季度交付，并在2023年进行量产，而英特尔3也有望在2023年底进入早期生产阶段。请注意，这些都是早期的图表，我们离Meteor Lake和Arrow Lake CPU的推出还有几年时间。
