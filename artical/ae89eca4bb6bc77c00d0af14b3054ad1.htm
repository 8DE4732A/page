微软宣布新增40个国家的PC Game Pass订阅 Insider今天开始试用

去年，微软透露计划将其PC Game Pass订阅服务带到官方支持的Xbox地区以外的国家。虽然第一次努力只涉及五个国家，但现在该公司正在将多达40个国家加入其支持名单。

对于那些不熟悉PC Game Pass的人来说，它是Xbox Game Pass的姐妹服务，有数百个PC游戏，包括微软自己的Xbox派对版本和贝塞斯达游戏，以及访问EA Play游戏。
