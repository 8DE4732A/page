微软：相信《使命召唤》能在NS上原生运行 XGP价格不会因为合并而上涨

今年2月份，微软宣布与任天堂签署10年协议，将把《使命召唤》带给任天堂玩家。如何让《使命召唤》在Switch运行呢？有人猜测可能以云游戏(cloud-streaming)形式登陆，或是会登陆Switch 2？


近日在针对英国监管机构补救措施通知的最新回应中，微软声称有信心让《使命召唤》系列游戏在Switch上原生运行，包括《使命召唤：战区》等。微软称《使命召唤：战区》支持2015年发布的显卡，所以这款200人吃鸡游戏没理由无法针对Switch进行优化。

官方表示：“动视开发团队在优化游戏性能以适应可用硬件功能方面有着悠久历史。为《使命召唤：战区》提供支持的游戏引擎已成熟，并已经过优化，可在各种硬件设备上运行，包括XboxOne和XSX。因此Switch也是没问题的。”


之后微软还列举了《Apex英雄》《毁灭战士》《堡垒之夜》《孤岛危机3》能在NS上运行的例子。声称除《使命召唤：战区》之外，其他《使命召唤》作品(如《使命召唤19：现代战争2》)也能利用技术及时优化，确保能在Switch上运行。

微软还表示，如果成功收购动视暴雪，Xbox Game Pass的价格不会上涨。


英国监管机构竞争与市场管理局上月发布了这笔拟议交易的临时调查结果，担心微软可能会在加入动视热门内容后，选择提高其Game Pass订阅服务的价格。

CMA表示：“订阅价格可以很容易地修改，一旦微软添加了像《使命召唤》一样受欢迎的内容，它可能就会有这样做的动机。”

然而，在对CMA调查结果的最新回应中，微软表示，由于交易正在进行，它不打算提高Game Pass的价格，并声称这样做会适得其反，因为它会导致订阅者下降。

“Game Pass的价格不会因为合并而上涨，而且肯定不会上涨到抵消动视游戏在游戏通行证上同步发布带来的实质性好处的程度。考虑到Game Pass将继续受到B2P(购买即玩)的限制，这一点尤其如此。”

微软随后指出，自从将B社内容引入Game Pass服务以来，其并未提高Game Pass的价格，微软在收购B社母公司ZeniMax Media后，获得了《上古卷轴》、《辐射》、《毁灭战士》、《耻辱》等系列的所有权。

“动视和微软的合并将产生双重边际利润效应的经典消除，因为微软将能够以机会成本收购这些游戏，并有动机更广泛地分发它们，并增加Game Pass的产量，以达到其对照水平。为了增加产量，微软需要以较低的质量调整价格提供Game Pass。

这正是微软在过去通过向Game Pass添加内容时所做的事情，例如ZeniMax交易会导致额外的内容，但不会增加 Game Pass的订阅价格。尤其是考虑到Game Pass用户对价格敏感，而Game Pass的价格增加将影响所有用户，包括那些不重视或不玩COD的用户。

Game Pass订阅者可以在玩一个月后随时取消订阅。由于CoD游戏每年发布一次，因此任何影响都将是短暂的，因为在几个月内对新版本COD失去热情的玩家将因较高的价格而离开。”
