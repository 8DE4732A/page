微软即将把基于ChatGPT的必应搜索框带入Windows 11任务栏

微软今天发布了Windows 11的一个大更新，将该公司新的人工智能驱动的必应搜索添加到任务栏。新的Windows 11更新将提供对新的必应聊天功能的快速访问，以及一堆新功能。Windows 11还对小工具进行了改进，有了更好的触摸模式，有了屏幕录制功能，有了记事本内的标签，还有更多。

新的必应整合是一个惊喜，微软还没有对其Windows Insiders进行测试。一个新的必应图标将出现在任务栏的搜索框中，微软在搜索栏中强调了新的聊天答案体验。虽然聊天答案不会直接出现在搜索栏中，但Windows 11用户将能够从这里快速启动Edge中的必应聊天--只要他们能够访问必应预览。

由人工智能驱动的必应将出现在Windows搜索中

"我们正在重新想象我认为的未来越来越多的人工智能驱动的Windows，"微软的消费者营销主管尤素福-迈赫迪在接受采访时说。"这是一个非常重要的功能，今天有超过5亿人经常使用搜索框。"

现在有100多万人在169个国家测试新的必应预览版，Windows负责人Panos Panay在一篇博文中说，"很快，数亿Windows 11用户可以获得这项令人难以置信的新技术，从他们的Windows任务栏上直接搜索、聊天、回答问题和生成内容"。这表明，微软正计划大大扩展新的必应预览。

将必应聊天扩展到Windows 11任务栏，是在微软在移动端和Skype对话中向必应推出相同模式后的一周。微软还准备很快在Office应用程序中演示其类似ChatGPT的人工智能。

不过，这个新的Windows 11大更新并不仅仅是关于必应的。微软还为iOS开放了Phone Link应用的预览，这意味着iPhone用户可以将他们的设备与Windows连接起来。这将包括发送和接收信息（是的，甚至是iMessage）、通话和通知的权限。你可以在这里阅读更多关于微软为iOS开发的新Phone Link应用。

Windows 11中新的第三方小工具

微软还在改进Windows 11中的部件系统，包括来自Meta和Spotify的第三方选项。由于有了新的平板电脑优化的任务栏，今天连Windows 11中的触摸体验也得到了改善。如果你有一个像Surface Pro这样的设备，断开键盘后，系统将折叠任务栏，使其更加紧凑，简化的任务栏是为触摸设计的。

Windows 11也终于有了一个官方的屏幕录制工具。虽然Xbox Game Pass已经提供了记录应用程序的能力，但Snipping Tool现在包括屏幕录制，让你捕捉整个屏幕或对应用程序进行裁剪。

Windows 11中的屏幕录制

在其他方面，微软也在改进其内置的团队体验，让人们更容易预览他们的视频，更快速地访问对话。Windows 11中的快速协助应用程序也经过重新设计，使其更容易帮助你可能需要提供技术援助的朋友和家人。

如果你是一个记事本粉丝，那么你可能会很高兴听到标签支持是这次Windows 11更新的一部分。微软还包括新的盲文显示支持，并将其语音访问功能带出预览版，同时还有新的切换功能来提高笔记本电脑的能源效率。

这些新的Windows 11功能今天都可以通过手动检查Windows Update的新更新来获得。微软表示，这些功能将在3月14日发布的3月2023日月度安全更新中广泛推出。
