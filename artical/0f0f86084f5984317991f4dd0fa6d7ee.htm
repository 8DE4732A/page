每五台电脑就有一台运行Windows 11 采用率达到历史最高水平

又到了月初，StatCounter的最新数据出来了，2023年3月的报告显示，Windows 11的增长持续且不可阻挡--微软最新的操作系统现在每五台Windows PC就有一台是最新版本，它的采用率几乎达到了21%。

在2023年3月，Windows 11获得了1.81%的份额，从19.13%增加到20.94%。然而，Windows 10也上升了，尽管没有像2023年2月那样剧烈，当时微软结束了Windows 7和8.1的支持，这一目前的市场领导者占了73.48%，比上个月增加了0.17%。

正如预期的那样，在微软宣布结束扩展安全更新计划（ESU）后，人们开始更快地离开Windows 7和8.1。Windows 7继续下滑，达到3.72%的关口，损失了1.62%。Windows 8.1只有微不足道的0.84%，并且还比2023年2月减少了0.3%。

根据Statcounter的数据，以下是Windows市场的情况：

Windows 10 - 73.48%

Windows 11 - 29.94%

Windows 7 - 3.72%

Windows 8.1 - 0.84

Windows 8 - 0.51

考虑到其他操作系统（Android、iOS、macOS、ChromeOS、Linux和其他），Windows仍然是第二大最受欢迎的操作系统，仅输给Android。微软的桌面操作系统在Android和iOS之间的市场份额为28.29%（+0.56点）。

Android - 41.56% (-2.32)

Windows - 28.89% (+0.56)

iOS - 17.29% (+0.16)

macOS - 7% (+0.33)

未知操作系统 - 3.05% (+0.62)

来自Statcounter 2023年3月报告的操作系统图表
