AMD锐龙9 7000X3D正式登场：144MB缓存 两大神级优化

去年，集成了3D V-Cache堆叠缓存的锐龙7 5800X3D大杀四方，深受游戏玩家青睐，意外成为AMD的一款卡爆品。今年初，AMD发布了基于Zen4架构的锐龙7000X3D系列产品，包括16核心的锐龙9 7950X3D、12核心的锐龙9 7900X3D、8核心的锐龙7 7800X3D，继续堆叠最多64MB 3D V-Cache缓存，总容量最高达惊人的144MB。

现在，锐龙9 7950X3D、锐龙9 7900X3D率先登场，明晚22点正式开卖，官方建议零售价分别为5299元、4499元，相比于标准版分别贵1300元、1200元。

相比之下，i9-13900KS目前需要多达5799元(前几天还是5999元)，但锐龙9 7950X3D的游戏性能可以轻松超越之，功耗还更低。

至于具体表现，请参考我们的首发评测。

锐龙7 7800X3D则将在4月6日跟进上市，价格预计会在3399元左右。

锐龙7 7800X3D和上代锐龙5 7800X3D一样，内部都是单CCX，直接堆叠缓存，没啥可说的。

锐龙9 7950X3D、锐龙9 7900X3D则是都有两颗CCX小芯片，64MB缓存仅堆叠在其中一个CCX之上，而不是平均分布。

AMD表示，堆叠缓存的CCX可以将延迟降低到最小程度，标准的CCX则适合高频率高性能的场景需求，二者结合可以获得一种绝佳的平衡。

相比双CCX都堆叠缓存的方案，这样可以大大降低成本，既能明显提升游戏性能，而对缓存不敏感、更需要高频率的应用可以在标准CCX上获得更好的表现。

当然，这种“非均衡”设计需要软件层面的优化，主板芯片组驱动需要升级到5.01.03.005版本或更高。

AMD为锐龙7000X3D系列准备了两大优化技术，一个是3D V-Cache性能优化驱动(3D V-Cache Performance Optimizer Driver)。

它会实时评估应用场景、负载和性能，动态调整Windows操作系统优先访问和使用的CCX与核心，需要大缓存的游戏就交给缓存CCX，需要高频率的其他应用就交给标准CCX。

这一过程是由驱动自动控制的，用户也可以在BIOS里手动切换缓存或频率，比如专心玩游戏的时候就选择缓存档。

另一项技术是AMD PPM Provisioning File Driver，它一直就有，这次加入了对锐龙9 7000X3D的特殊支持和优化。

当检测到游戏运行的时候，会自动将其固定在一个CCX上(几乎都会选择缓存CCX)，另一个CCX则进入休眠状态，从而保证足够的游戏性能。

如果游戏的同时还在执行其他任务，比如直播什么的，线程利用率足够高，超过了一个CCX的承载能力，就会自动启动休眠的第二个CCX，兼顾多任务性能。

驱动安装成功后，就可以在设备管理的系统设备里看到以上两个项目，这代表可以发挥正常性能了。

系统服务中也可以查看AMD 3D V-Cache Performance Optimizer Service的运行状态，需要是自动启动、正在运行。

任务管理中能看到名为amd3dvcacheSvc.exe的系统进程，以及名为amd3dvcacheUser.exe的用户进程。

《看门狗：军团》、《银河破裂者》、《F1 2021》、《杀出重围：人类分裂》、《全面战争：三国》这些游戏都可以获得明显的提升，幅度均超过20％，最高甚至达到41％。

《异域骑兵》、《CSGO》反而还会出现一些倒退，需要进一步优化。

锐龙7 5800X3D是不允许超频的，锐龙7000X3D系列也不能自由超外频和倍频，但官方仍提供了一些超频选项。

一是Curve Optimizer，二是PBO，都可以在Ryzen Master或者主板BIOS中开启，一键自动优化性能。

另外，内存超频也是支持的，无论Intel XMP 3.0，还是AMD EXPO，抑或其他内存超频技术，都可以。

按照官方数据，锐龙9 7950X3D打开Curve Optimizer之后，多核性能可提升约2％， 如果再打开PBO，提升幅度可以来到2.4％。

最后一个问题，堆叠如此之多的缓存，功耗并不会失控，锐龙7000X3D系列的热设计功耗设定在120W。

根据官方数据，锐龙9 7950X3D在跑CineBench R23多核测试的时候，整机功耗仅为101W，相比于锐龙9 7950X还降低了大约20W，因此能效提升了13％。
