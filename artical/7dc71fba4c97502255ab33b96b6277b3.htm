微软与Parallels达成合作 在苹果M1和M2 Mac上支持Windows 11

与Parallels达成合作后，微软正式支持Windows 11操作系统在苹果的M1和M2 Mac上的虚拟环境中运行。根据今天发布的新的微软支持文章，Parallels Desktop 18版本是在苹果最新的M1和M2 Macs上运行Arm版本Windows 11的"授权解决方案"。

虽然微软对支持哪些CPU来运行Windows 11有限制，但该公司很乐意让苹果M1和M2用户在虚拟机中运行操作系统，但这不会提供通过原生解决方案所能达到的最佳性能，苹果公司方面也没有显示出在M1或M2 Mac上支持Boot Camp的迹象。

尽管如此，微软计划正式授权Parallels支持这种在苹果最新的Mac上运行Windows 11的方式，这比我们目前所拥有的更进一步。到目前为止，微软只将Arm的Windows版本直接授权给OEM厂商，使得M1和M2用户很难在虚拟机中正式运行它。

Parallels从Parallels Desktop 16.5版本开始支持M1芯片上的Windows，这个最新版本可以让用户一键下载并安装Windows 11。Parallels还通过与Apple芯片配对的虚拟TPM处理Windows 11的TPM和安全启动要求的复杂性。

虽然在技术上可以购买Windows 10许可证密钥并激活Arm版本，但在这之前并没有正式授权。对于可能拥有混合使用Windows和macOS的笔记本电脑的Windows企业用户来说，这是一个更大的法律上的问题。

图为Mac硬件上运行的虚拟机中运行的Windows 11

不过，这里的许可情况还不是很清楚。Parallels说企业可以"通过正常的Windows采购流程"购买Windows 11许可证，而个人可以直接从微软获得199美元的Windows 11专业版许可证。但是不清楚微软是如何改变其许可规则的。

微软还为那些想在M1或M2 Mac上访问Windows 11的人提供另一种选择。Windows 365 Cloud PC。这是托管在云端的虚拟机，它包括完全的应用兼容性。在Parallels Desktop 18中运行的Arm版本的Windows 11对支持的硬件、游戏和应用程序的类型有一些限制。硬件驱动程序只有在专门为Windows 11 Arm-based PC设计的情况下才能工作。某些游戏也不能运行，如果它们使用OpenGL 3.3或更高版本或依赖反作弊驱动程序。一些应用程序也不能正常运行，包括那些试图集成到Windows外壳中的应用程序，如云存储应用程序或辅助技术。同样地，一些第三方杀毒软件也无法安装。

对于其他一切，Windows on Arm现在有x64应用程序模拟，所以大多数应用程序应该运行良好。
