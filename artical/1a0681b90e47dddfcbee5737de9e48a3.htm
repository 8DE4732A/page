每五台电脑中就有一台运行Windows 11 结束支持推动新版系统份额大涨

在每个月初，Statcounter都会发布报告，详细介绍最新的浏览器和操作系统市场的转变。2023年2月的数据现在已经出来了，它显示了几个有趣的变化，比如Windows 11接近20%的里程碑，而Windows 10的市场份额意外大涨。

根据Statcounter的数据，2023年2月，Windows 11的市场份额增加了1.01个百分点，达到了19.13%的历史新高。换句话说，几乎每五台电脑就有一台运行微软最新的操作系统。2023年2月28日发布的最新功能更新，可能会诱使更多的Windows 10用户迁移到Windows 11。

操作系统市场是巨大的，很少经历突然的高峰或重大变化。然而，2023年2月是一个例外。2023年1月，微软让Windows 7和8.1彻底退出市场，并结束了扩展安全更新计划。这导致更多的公司快速行动起来，将他们的电脑升级到Windows 10，使其市场份额从1月的68.86%上升到2月的73.25%。

尽管最后结束了支持，Windows 7仍然是第三大最受欢迎的Windows版本。然而，它失去了一大块用户，从9.55%下降到5.39%。Windows 8.1的市场份额也从2.28%减半到1.15%。

Windows 10 - 73.25% (+4.39)

Windows 11 - 19.13% (+1.01)

Windows 7 - 5.39% (-4.16)

Windows 8.1 - 1.15% (-1.13)

Windows 8 - 0.52% (-0.1)
