Windows 11在Steam玩家中拥有接近29%的使用比例

每个月，Valve都会进行一次自愿调查，以收集关于其客户用来访问Steam的PC配置的信息。这些数据可以帮助开发者优化他们的游戏，以达到尽可能多的玩家，并在不同的技术上做出明智的投资。Steam硬件和软件调查也是关于各种Windows版本及其在游戏玩家中的成功的重要信息来源。

参与调查是可选的，所以数据并不代表100%的Steam用户。另外，Steam官方页面比较的是2022年10月的最新数据，而不是2022年11月的数据，因此本文和官方报告之间有时间上的距离。

调查结果显示，Windows 11不断从Windows 10手中抢夺用户。根据Valve的数据，在2022年12月，Windows 11达到28.42%，与2022年11月相比增加了0.44个百分点。

Windows 10 64位是Steam上最主要的操作系统品种，在2022年12月只损失了0.18个百分点，这款已经有8岁的操作系统现在拥有65.42%的市场份额。Windows 7 64位经历了类似的下降。Valve表示，Steam上第三大最受欢迎的Windows版本目前只剩下1.66%（-0.22%）。

Windows 10 64位 65.42% (-0.18%)

Windows 11 28.42% (+0.44%)

Windows 7 64位 1.66% (-0.22%)

Windows 8 64位 0.41%

Windows 7 32位: 0.11%

总体而言，微软的操作系统家族以96.15%的市场份额（+0.04%）牢牢占领绝对优势地位，苹果以2.48%（+0.03）位居第二，Linux以1.38%（-0.06）位居第三。
