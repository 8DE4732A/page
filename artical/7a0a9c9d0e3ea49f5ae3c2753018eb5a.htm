微软抛弃了Windows 8时代的设计 努力实现操作系统界面现代化

Windows 11在2023年有几个小幅功能更新，现在我们已经从开发频道的预览版中瞥见了一些可能正在开发的界面变化。据熟悉开发情况的消息人士称，微软设计师提出了几项改变，以取代操作系统内的弹出式提示框。

你可能知道，Windows弹出式提示框最后一次刷新是在Windows 8。Windows 10甚至是Windows 11的一些功能都依赖于Metro时代的设计。

微软希望用WinUI和Windows 11的现代外观取代所有剩余的Windows 8和Windows 7时代的弹出窗口。如下面的截图所示，微软正在测试Windows 11中所有弹出式提示框的新外观。这些弹出框现在使用WinUI，它们也有圆角。

Windows防火墙/安全弹出窗口在预览版中以新的外观刷新，但它们是隐藏的

当然，弹出框内的按钮也是圆角的，看起来更现代，而且确实更好看。作为现代化的一部分，微软正在用更常见的颜色取消岩石般的重点主题背景，如深色模式下的深灰色或浅色模式下的白色。

Windows 11中的一个现代对话框

在其中一项提议中，微软指出需要"简化设计"，并更多地关注微小的细节。

微软已经开始测试Windows防火墙、"重命名此电脑"、"重置此电脑"等弹出式对话框的现代外观。设置应用程序中的更多盒子使用了现代界面，支持黑暗主题、圆角等。刷新的"回到早期构建"对话框在Windows 11上看起来也很不错。

当然，"重命名这台电脑"或"重置这台电脑"的页面并不经常被用户访问，但这次刷新是操作系统现代化的另一个重要步骤。

WinUI和圆角似乎是Windows 11和其他微软产品的主要主题。与以往一样，未来几年可能会有很多变化，因为Windows一直在开发中。事实上，据报道，微软已经在考虑Windows 12。

在生产构建中，我们已经看到UI的主要元素，包括控制面板、上下文菜单、通知中心和使用WinUI的按钮。值得注意的是，在过去的三四年里，流畅设计、WinUI和圆角的设计理念一直在流传。

Windows 11的下一次重要更新预计将在2月或3月出现，而且据信微软也在为2023年秋季的重大功能更新做准备。
